# **MINA2000 "ElectroCute"**

`ElectroCute` is the *MINAv2* reference implementation. It's a 5-stage pipelined design with separate instruction and data caches.

## **Progress**
- Implemented 5-stage pipeline
- Implemented operand forwarding
- Implemented ADD, SUB, MOVH, ADR
